Explore Cascadia

Explore Cascadia is a webpage where users can select a State Park within Cascadia. Once they select a park, they are given a five day weather forecast, park directions, park hours, a phone number for the office at the park, and two images of the park.
This project required extensive work with multiple web API's, jQuery, JavaScript, and utilizing the Chrome Developer tools for styling and debugging. This project also required learning a new framework, Semantic UI.
